Department Network using VLAN

📌 Overview

This project demonstrates a multi-floor network where departments are separated using VLANs and connected using trunking and inter-VLAN routing.

🧱 VLANs Used

* VLAN 10 – HR
* VLAN 20 – IT
* VLAN 30 – Finance
* VLAN 40 – Admin
* VLAN 50 – Marketing
* VLAN 60 – Management

🛠 Tools

* Cisco Packet Tracer

✅ Result

Successful communication between all VLANs using routing.
